# projeto-spotify
 O objetivo deste projeto é criar uma aplicação web que simule as funcionalidades básicas do Spotify, como reprodução de músicas, criação de playlists, busca de artistas e álbuns, e autenticação de usuários. A aplicação será dividida em duas partes principais: o frontend (interface do usuário) e o backend (lógica de negócios e banco de dados).
